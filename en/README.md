# Dagobang Help Center

Dagobang is a non-custodial browser extension built for meme trading. It overlays a floating trading panel on top of the chart/order pages you already use, so you can trade without switching tabs, and it improves execution for fast trading via silent signing and multi-endpoint broadcasting.

## Key Features

- Low fees: cold start at 0%, future plan ≤ 0.1% cap (subject to production config)
- Non-custodial: private keys / mnemonics are encrypted and stored locally only
- Floating panel trading: trade directly on sites like Fourmeme / GMGN / Axiom
- Limit orders & auto execution: trigger buys/sells at target prices and record results
- Advanced Auto-Sell: multi-step take-profit / stop-loss, optional trailing stop
- Privacy & execution: Protect RPC, Anti-MEV, multi-endpoint racing, optional bloXroute private broadcast
