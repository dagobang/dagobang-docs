# 安装与快速开始

本页面向第一次使用打狗棒的用户，覆盖安装、创建/导入钱包、打开悬浮窗并完成第一笔交易的最短路径。

## 1. 安装插件

打狗棒扩展基于 WXT 构建，构建产物可直接以“加载已解压的扩展程序”的方式安装。

1. 在 `dagobang-extention/` 目录执行：
   - `npm install`
   - `npm run build`
2. 构建产物位于：`dagobang-extention/.output/chrome-mv3/`
3. 打开浏览器扩展管理页：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
4. 打开“开发者模式”，点击“加载已解压的扩展程序”，选择 `.output/chrome-mv3/` 目录。

可选：执行 `npm run zip` 会输出 zip 包，解压后同样可按上述方式加载。

## 2. 第一次打开：创建或导入钱包

打开插件 Popup（浏览器工具栏里的扩展图标）后，首次会看到欢迎页：

- 创建钱包：设置密码（至少 6 位）→ 创建 → 备份助记词
- 导入钱包：设置密码（至少 6 位）→ 粘贴助记词或私钥 → 导入 → 备份（如有返回助记词）

建议：

- 密码用于本地加密与解锁，不会上传
- 助记词/私钥只建议离线备份，不要在聊天工具里传播，不要截图公开

## 3. 解锁与自动锁定

- 解锁：Popup 输入密码解锁
- 自动锁定：默认 30 分钟自动锁定（可在 Popup → Settings 调整）

限价单与自动触发依赖“钱包处于解锁态”。钱包锁定时，限价单不会自动执行。

## 4. 打开悬浮窗

安装后，当你访问支持的看盘/挂单页面时，页面会出现打狗棒悬浮窗（可拖拽、可最小化为圆形图标）。

如果悬浮窗提示未解锁，可点击“解锁”，它会自动拉起 Popup 进行解锁。

## 5. 完成第一笔交易（建议流程）

1. 在 Popup → Settings 完成基础配置：
   - 配置 `RPC URLs`（读链/报价/查询）
   - 如要开启 `Anti‑MEV`，先配置至少 1 条 `Protected RPC URLs`（用于广播交易）
2. 回到看盘页面，在悬浮窗中选择一个买入金额预设（单位 BNB）并下单。
3. 首次卖出前，如出现需要授权的提示，先执行 Approve 授权（悬浮窗内有按钮）。
