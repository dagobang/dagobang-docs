# 安全、隐私与免责声明

本章只讲三件事：

- 打狗棒如何处理你的密钥与交易（安全与隐私边界）
- 你在使用中应当遵守的最小安全实践
- 风险与免责声明

## 1. 非托管与本地加密

- 私钥/助记词仅在本地保存，并由你设置的密码加密
- 插件不会把你的密钥上传到服务器
- 解锁后，插件会在本地内存中使用签名能力完成交易签名，并将“已签名交易”广播到你配置的节点

你应该默认：

- 任何能获取你明文助记词/私钥的人，都能完全控制资金
- 任何能控制你浏览器环境（恶意扩展、木马、远程控制）的人，都可能间接窃取资产

## 2. 风险提示（Meme 交易特有）

- 高波动：滑点、回撤、插针与流动性枯竭都可能导致超预期亏损
- 高对抗：夹子、三明治攻击、MEV 竞争会影响成交价与成交概率
- 合约风险：恶意税、黑名单、转账限制、蜜罐等会造成无法卖出或高额损失
- 节点风险：RPC 不稳定会导致报价滞后、交易广播失败或延迟

## 3. 最小安全实践（强烈建议）

- 密码：设置独立强密码，不复用其他平台密码
- 备份：助记词只做离线备份（纸质/硬件），不要截图、不要网盘、不要聊天工具传输
- 环境：只在可信设备与浏览器配置下使用，避免安装来路不明的扩展
- 资金：建议小额测试，分仓管理，避免把全部资金放在同一地址
- 节点：Protected RPC 使用可信渠道提供的节点，避免不明节点记录/审计你的交易

## 4. 免责声明

- 打狗棒仅提供工具能力，不构成投资建议
- 你需自行承担交易损失、节点故障、合约风险、市场波动等全部风险
- 任何“收益保证/包赚/内幕”均与打狗棒无关
