# 悬浮窗交易面板

悬浮窗是你在看盘页面里最常用的交易入口：无需切换页面，直接完成买入、卖出、授权与常用执行参数的切换。

## 1. 基本操作与窗口行为

- 拖拽移动：按住顶部栏拖动
- 最小化：右上角按钮将面板最小化为圆形图标（点击图标可恢复）
- 解锁提示：如果钱包处于锁定状态，面板会显示遮罩层；点击后会自动打开 Popup 解锁
- 支持链：当前主链为 BSC（chainId 56）

## 2. 顶部栏快捷入口

顶部栏从左到右包含这些快捷入口：

- 限价挂单面板：打开/关闭“限价挂单与订单列表”
- RPC 面板：查看当前 RPC / Protect RPC 列表并测速
- Daily Analysis：查看地址的按日盈亏曲线（数据源来自第三方接口）
- 预设编辑：铅笔进入编辑模式，勾选保存预设
- 最小化：将面板收起

## 3. 买入（Buy）

买入区以 BNB 计价（预设按钮为常用金额）：

- 余额展示：显示当前地址的 BNB 余额
- 金额预设：点击预设按钮后直接按该金额发起买入
- 执行模式：
  - Default：按常规流程提交交易
  - Turbo：为更贴合高频的执行路径做优化（更依赖你对链上交易与风险的理解）
- Gas 档位：在 Slow / Standard / Fast / Turbo 之间循环切换（具体 gwei 可在 Popup → Settings 设置）
- 滑点：在 30% / 40% / 50% / 90% 之间循环切换（对应 bps：3000/4000/5000/9000）
- 高级自动卖出：可在买入区开启并配置“买入成交后自动创建卖出挂单”，详见 [限价挂单与高级自动卖出](automation.md)

提示：

- 买入成功后，插件会尝试为该代币执行“卖出授权（Approve）”的预处理（如需要）
- 交易执行过程中会有 Toast 提示，并展示本次广播来源（例如某条 RPC 或 bloXroute）

## 4. 卖出（Sell）

卖出区以“卖出仓位百分比”计价：

- 余额展示：显示当前代币余额（简化显示）
- 百分比预设：10% / 25% / 50% / 100%（可编辑）
- Approve：首次卖出前可能需要授权代币给路由合约（面板会提供按钮）
- 卖出提交：点击百分比后发起卖出

与自动化联动：

- 当你执行 100% 卖出时，插件会尝试取消该代币下的所有限价单（避免“清仓后仍有挂单”）

## 5. 编辑买入/卖出预设

点击顶部栏的铅笔进入编辑模式：

- 可编辑买入预设（BNB）与卖出预设（%）
- 点击勾选保存；保存后会写入本地 Settings
